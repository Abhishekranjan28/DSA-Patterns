# DSA Tree

A responsive, data-driven DSA roadmap built with React, TypeScript, and Tailwind-ready CSS.

## Run locally

```bash
npm install
npm run dev
```

Use `npm run build` for a production verification build.

## Content architecture

- `src/data/roadmap.ts` contains the complete navigable taxonomy, TypeScript data types, authored learning modules, templates, and practice-problem metadata.
- `src/App.tsx` contains the reusable dashboard, roadmap explorer, detail drawer, practice interactions, and dry-run visualizations.
- Progress is stored in the browser under `dsa-tree-progress` using localStorage.
